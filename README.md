# CGCSG

A playground for visualising Signed Distance Functions and Constructive Solid Geometry using Raymarching in C++.

## Requirements
* `CMake version 3.9 or greater` 
* `SDL version 2`
* `GLM 0.9.9 or greater`

Optionally:
* `OpenMP 2.0 or greater`
